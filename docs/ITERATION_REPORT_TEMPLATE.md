# ITERATION_REPORT_TEMPLATE.md

Используй этот шаблон в описании PR или финальном отчёте агента.

```md
## Что сделано

- ...

## Зачем это сделано

- ...

## Какие файлы изменены

- `path/to/file` — что изменилось.

## Как проверить

1. ...
2. ...
3. ...

## Документация

- [ ] Обновлён `docs/TASKS.md`.
- [ ] Обновлён `docs/CHANGELOG.md`.
- [ ] Обновлён `docs/PROJECT_STATE.md`.
- [ ] Обновлён `docs/SCENARIO.md`, если менялся сценарий.
- [ ] Обновлён `docs/DECISIONS.md`, если менялась архитектура.
- [ ] Обновлён `docs/CREDITS.md`, если добавлялись ассеты.

## Риски

- ...

## Что делать дальше

- ...
```

## Правило

Если пункт не относится к задаче, нужно написать `не требуется`, а не удалять раздел без объяснения.
