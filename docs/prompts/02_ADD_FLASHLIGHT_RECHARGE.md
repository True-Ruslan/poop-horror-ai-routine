# Prompt 02 — Add Flashlight Recharge Point

Прочитай:

```text
AGENTS.md
docs/CONTEXT.md
docs/TASKS.md
```

Задача:

Добавь систему зарядки фонарика от розетки.

Требования:

1. Создай сцену:

```text
game/scenes/objects/PowerOutlet.tscn
```

2. Создай скрипт:

```text
game/scripts/objects/PowerOutlet.gd
```

3. Розетка должна быть интерактивной.
4. При взаимодействии она должна находить фонарик игрока и зарядить его на настраиваемое значение.
5. Добавь одну розетку в демо-сцену.
6. Обнови документацию.

Ограничения:

- Не добавляй инвентарь.
- Не добавляй батарейки как предметы.
- Не меняй архитектуру InteractionRaycast без необходимости.

Definition of Done:

- Можно подойти к розетке.
- Появляется prompt.
- При нажатии E заряд фонарика увеличивается.
- Проект запускается без ошибок.
