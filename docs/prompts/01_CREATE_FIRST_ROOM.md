# Prompt 01 — Create First Custom Room

Прочитай:

```text
AGENTS.md
docs/CONTEXT.md
docs/TASKS.md
docs/DESIGN_GUIDE.md
```

Задача:

Создай новую сцену:

```text
game/scenes/levels/FirstRoom.tscn
```

Требования:

1. Не удаляй `TemplateLevel.tscn`.
2. Создай небольшую комнату 10x10 метров.
3. Добавь игрока.
4. Добавь одну дверь.
5. Добавь одну записку.
6. Добавь один выключатель света.
7. Добавь один простой horror event через существующий `HorrorEventManager`.
8. Обнови `Main.tscn`, чтобы он запускал `FirstRoom.tscn`.
9. Обнови `docs/TASKS.md`.
10. Обнови `docs/CHANGELOG.md`.

Ограничения:

- Не добавляй инвентарь.
- Не добавляй врага.
- Не добавляй сторонние ассеты.
- Не меняй PlayerController без необходимости.

Проверка:

- Проект запускается.
- Игрок может ходить.
- Можно прочитать записку.
- Можно щёлкнуть выключатель.
- Можно взаимодействовать с дверью.
