# SCENARIO_TEMPLATE.md

Скопируй этот файл как:

```text

```

И заполни под свою игру.

## 1. Название

Рабочее название игры:

```text
Ночное дежурство
```

## 2. Жанр

```text
Короткий психологический хоррор от первого лица.
```

## 3. Референсы

```text
- Fears to Fathom — бытовая цель, ощущение реальности, чужое присутствие.
- Trash Horror — короткий формат, VHS/retro-атмосфера.
- Layers of Fear — пространство отражает психику героя.
```

## 4. Логлайн

```text

```
