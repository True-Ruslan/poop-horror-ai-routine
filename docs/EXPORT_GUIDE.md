# EXPORT_GUIDE.md

## Цель

Этот файл описывает путь к сборке игры для Windows/Linux/macOS.

## Минимальный путь для Windows

1. Открыть проект в Godot.
2. Скачать Export Templates для своей версии Godot.
3. Открыть:

```text
Project → Export
```

4. Добавить preset:

```text
Windows Desktop
```

5. Указать папку:

```text
builds/windows/
```

6. Нажать Export Project.

## Что проверить перед сборкой

- Игра запускается из редактора.
- Главное меню или стартовая сцена работает.
- Мышь захватывается при старте.
- Esc не ломает управление.
- Нет ошибок в Output.
- Все используемые ассеты есть в проекте.
- В `docs/CREDITS.md` записаны все сторонние ресурсы.
