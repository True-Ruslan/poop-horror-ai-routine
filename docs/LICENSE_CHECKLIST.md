# LICENSE_CHECKLIST.md

Перед добавлением любого стороннего ресурса проверь этот список.

## Можно обычно использовать

- MIT
- BSD
- Apache-2.0
- CC0
- CC-BY, если указана атрибуция

## Использовать осторожно

- GPL-3.0
- LGPL
- CC-BY-SA
- Asset Store licenses с ограничениями
- AI-generated assets без понятных commercial rights

## Не использовать без отдельного решения

- Non-commercial licenses
- Editorial use only
- Personal use only
- No derivatives
- Неизвестная лицензия
- Файлы из чужих игр
- Музыка/звуки из YouTube без лицензии
- Модели с Sketchfab без проверки условий

## Что записывать в CREDITS.md

- Название.
- Ссылка на источник.
- Автор.
- Лицензия.
- Дата добавления.
- Где используется.
- Нужно ли указывать автора в игре.

## Минимальное правило

Если ты не можешь объяснить, почему ресурс можно использовать в коммерческой игре, не добавляй его в проект.
